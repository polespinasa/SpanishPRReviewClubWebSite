RPC: Es un protocolo JSON-based en HTTP, es como usualmente interactuamos con el server de Core.

IPC: Inter-Process Communication. Habilita la comunicación entre procesos y es la base para migrar la actual arquitectura monolítica de bitcoind a multiprocess.

Interfaces de Core: Son classes abstractas en C++, por ahora la mayoría se usan internamente en un monolito pero con libmultiprocess puedes interactuar con otro proceso como si estuvieran en el mismo.
